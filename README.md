Project for my CMPT 432 - Compilers Class at Marist University.

How to use:
npm start -- [--quiet | --debug] [filePath]
Quiet mode hides the token traces and just shows lex results, debug mode shows them all

Example for quiet mode:
npm start -- --quiet test-files/testStep6.txt

Example for debug mode:
npm start -- --debug test-files/testStep6.txt
